COMPUTER SCIENCE

Computer science is the study of computers, algorithms, and computational systems. It focuses on understanding how computers work, solving complex problems, and developing software and applications. It encompasses various fields such as programming, artificial intelligence, data structures, networking, and cybersecurity, making it integral to advancements in technology and innovation.

PARTS OF COMPUTER


Monitor: The main part of a computer system that displays the output. CPU: The central processing unit that performs calculations and executes instructions. Keyboard: An input device used by the user to send input data or messages to the computer. Mouse: An input device used by the user to move a cursor and select items on the screen. Storage Device: A device that stores data and programs, such as a hard disk drive or a solid state drive. Computer Memory: A device that holds data and programs temporarily, such as random access memory or RAM. UPS: An uninterruptible power supply that provides backup power in case of power failure. Motherboard: The main board that holds and connects all of the other components.

Monitor : A monitor is an electronic output device used to display information


Mouse : A computer mouse is a handheld hardware input device that controls a cursor in a GUI


Keyboard : A keyboard is an input device used to enter text, numbers, and commands into computer


CPU : The CPU is used for processing instructions from programs and components


Speaker : A speaker in a computer is an output device that produces sound


RAM : It is volatile memory that temporarily stores data enabling quick read and write access.


ROM : It is a type of non-volatile memory used in computers to store data permanently.


Processor : A processor (CPU) is an electronic circuit that executes instructions to perform tasks


Capsitor : A capacitor is component that stores and releases electrical energy in the form of an electric field.


Tansistor : A transistor is a semiconductor device used to amplify or switch electronic signals.
